
## 목표

- 채팅서버 구축
- STOMP를 활용하여 여러 채팅방 쉽게 처리하게 만들기
- Kafka를 이용하여 다수의 인스턴스에서도 동작 가능하도록 가용성 확보


## 사용기술
백엔드 : Spring boot

통신 : STOMP

메세지 큐 : Kafka
